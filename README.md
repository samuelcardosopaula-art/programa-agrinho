# Projeto Agrinho 2026

## Tema

Conexão Campo e Cidade

## Descrição

Este projeto foi desenvolvido para o concurso Agrinho 2026 com o tema "Conexão Campo e Cidade".

O objetivo é mostrar como o campo e a cidade dependem um do outro. O campo produz alimentos, matérias-primas e recursos essenciais, enquanto a cidade oferece tecnologia, serviços e oportunidades que contribuem para o desenvolvimento rural.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
